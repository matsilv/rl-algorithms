# rl-examples
Tensorflow implementation of some Reinforcement Learning algorithms.


### Algorithms

- [x] Deep Q-learning
- [x] Dueling architecture
- [x] Double Q-learning
- [x] Reinforce
- [x] Advantage Actor Critic (A2C)


### Usage
`python --help` for command usage.


### Requirements
- tensorflow 1.12.0
- gym 0.12.0
- numpy 1.17.0
- opencv-python 4.1.0
